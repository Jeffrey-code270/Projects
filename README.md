# Jeffrey Code Portfolio

This repository contains multiple projects showcasing different technologies and skills.

## Projects

### 1. PDF Analysis DevOps Project
**Location:** `pdf-analysis-project/`

A DevOps project that processes PDF documents to extract and analyze keyword frequencies, storing results in a PostgreSQL database.

**Technologies:** Python, PostgreSQL, Docker, NLTK, SQLite

**Features:**
- PDF text extraction and processing
- Keyword frequency analysis
- Database storage with conflict resolution
- Docker containerization
- Local SQLite testing mode

### 2. Portfolio Website
**Location:** `portfolio-website/`

A full-stack web application showcasing projects and skills with modern web technologies.

**Technologies:** React, Node.js, Express, MongoDB, Tailwind CSS, Vite

**Features:**
- Responsive React frontend
- RESTful API backend
- Contact form functionality
- Modern UI with Tailwind CSS
