# Portfolio Website

A full-stack web application showcasing Jeffrey's projects and skills.

## Technologies
- **Frontend:** React, Vite, Tailwind CSS
- **Backend:** Node.js, Express, MongoDB
- **Deployment:** Full-stack web application

## Setup

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm start
```

## Project Structure
- `frontend/` - React application with Vite
- `backend/` - Node.js Express server
- `backend/models/` - Database models
